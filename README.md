## Neurolinguals_NIPS
Repo of scripts for the 2024 NIPS [paper](https://openreview.net/pdf?id=t4CnKu6yXn) _"Investigating the role of modality and training objective on representational alignment between transformers and the brain"_.

Structure of this repo:
```
Neurolinguals_NIPS
└── scripts
    └── Model
        ├── Reps_Extraction
        └── RDM
    └── Brain
        ├── data_download
        ├── ROI_RDM
        └── Searchlight_RDM
    └── RSA
        ├── ROI_RSA
        └── Searchlight_RSA
    └── Viusalization
└── results
    ├── Model_RDM
    ├── Brain_ROI_RDM
    ├── Searchlight_RDM
    ├── Brain_ROI_RSA
    └── Searchlight_RSA
└── utlitis
```
